About Me

A passionate and skilled AWS Solution Architect, Ethical Hacker, and Cybersecurity Analyst with comprehensive expertise in cloud infrastructure design, penetration testing, vulnerability assessments, and security implementation. I have completed and been certified by various organizations for AWS Cloud Architecture, Security Engineering, and Ethical Hacking methodologies.

My expertise spans across three critical domains: designing scalable AWS architectures following Well-Architected Framework principles, conducting comprehensive penetration testing using industry-standard tools like Burp Suite, Nmap, and Metasploit, and performing detailed cybersecurity analysis including threat hunting, incident response, and security compliance auditing. I excel at integrating security-by-design principles into cloud architectures while identifying and mitigating vulnerabilities through systematic testing approaches.

With a Master's degree in IT Security and Big Data, I bring both theoretical knowledge and practical experience in OWASP Top 10 vulnerabilities, social engineering simulations, AWS security services, and automated security testing integration. I am committed to continuous learning and professional growth, seeking opportunities to leverage my tri-fold expertise in cloud architecture, ethical hacking, and cybersecurity analysis.

What I'm Doing

AWS Solution Architect

Ethical Hacker
